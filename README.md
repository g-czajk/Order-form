## Instructions on running locally

**Node version:**
`
  v14.15.3
` \
**npm version:**
`
  6.14.9
`

**Clone this repo:**

```
  git clone https://github.com/g-czajk/Order-form.git
```
**Go to the project directory:**

```
   cd Order-form
```
**Install dependencies:**

```
   npm install
```
**Start dev server:**

```
   npm start
```
